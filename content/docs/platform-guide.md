## 3. The AEO Editor

The Editor is divided into two modes:

### Focus Mode (Drafting)

Minimalist. Just you and the text.

- **Slash (`/`) commands** active.
- **AI Co-pilot (`Ctrl+Space`)** active.

### Engineer Mode (AEO)

Toggle the **"AEO Sidebar"** (Right Panel) to see:

- **Structure Check:** Are you missing a "Quick Answer"?
- **Schema Preview:** See the JSON-LD we are generating.
- **Canonical Status:** "Primary: Blog. Synced: Dev.to, Medium."

## Shortcuts Keymap

Move fast without touching the mouse.

| Combo          | Action                                      |
| :------------- | :------------------------------------------ |
| `G` then `D`   | Go to Dashboard                             |
| `G` then `I`   | Go to Intelligence                          |
| `G` then `A`   | Go to Articles                              |
| `Ctrl + K`     | Global Search (Prompts, Articles, Settings) |
| `Ctrl + Enter` | Deploy (Publish)                            |
| `Ctrl + Space` | AI Auto-complete                            |

## Mobile "Triage" Mode

The Texavor mobile web app is designed for **Triage**, not writing.

- **Approve/Reject** AI Topic suggestions.
- **Check Freshness Alerts.**
- **View Analytics.**
